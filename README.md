# Widgets Host

This website will host various widgets that will be used to be embedded into other web pages or Notion.

## Usage

### Retro Flip Clock
This widget displays a retro-style flip clock.
- **File**: `retro-flip-clock.html`
- **Features**: Real-time clock, smooth 3D flip animation, completely self-contained.
- **Customization**: Uses CSS variables for easy theming.
