# Learning Paths

## Overview
Structured roadmaps to help team members level up their skills.

## 1. Frontend Specialist
**Goal**: Master React codebase and internal state management.

### Roadmap
1.  **Level 1**: React Core
    *   *Read*: [React Docs (Beta)](https://beta.reactjs.org/) - Read "Thinking in React".
    *   *Practice*: Build a functional Todo app using only Hooks.
2.  **Level 2**: TypeScript
    *   *Read*: [TypeScript Handbook](https://www.typescriptlang.org/docs/).
    *   *Course*: Total TypeScript (Matt Pocock).
3.  **Level 3**: State & Performance
    *   *Topic*: Context API vs Redux vs Zustand.
    *   *Topic*: Profiling with Chrome DevTools.

## 2. Backend Specialist
**Goal**: Build scalable, secure APIs.

### Roadmap
1.  **Level 1**: API Design
    *   *Read*: "Design APIs for Humans".
    *   *Practice*: Build a CRUD API with Express/FastAPI.
2.  **Level 2**: Database Mastery
    *   *Read*: "DDIA (Designing Data-Intensive Applications)" - Chapters 1-3.
    *   *Topic*: Indexing strategies, Normalization.
3.  **Level 3**: Distributed Systems
    *   *Topic*: Caching (Redis), Message Queues (RabbitMQ/Kafka).

## 3. Recommended Books
-   *Clean Code* (Robert C. Martin)
-   *The Pragmatic Programmer* (Thomas & Hunt)
-   *Designing Data-Intensive Applications* (Martin Kleppmann)
