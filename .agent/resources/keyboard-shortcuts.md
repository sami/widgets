# Keyboard Shortcuts Cheat Sheet

## VS Code

| Action | macOS | Windows/Linux |
|---|---|---|
| **Command Palette** | `Cmd+Shift+P` | `Ctrl+Shift+P` |
| **Go to File** | `Cmd+P` | `Ctrl+P` |
| **Multi-Cursor** | `Opt+Click` | `Alt+Click` |
| **Format Document** | `Shift+Opt+F` | `Shift+Alt+F` |
| **Toggle Terminal** | `Ctrl+`` | `Ctrl+`` |
| **Duplicate Line Down** | `Shift+Opt+Down` | `Shift+Alt+Down` |

## Terminal (Zsh/Bash)

| Action | Shortcut |
|---|---|
| **Clear Screen** | `Ctrl+L` |
| **Go to Start of Line** | `Ctrl+A` |
| **Go to End of Line** | `Ctrl+E` |
| **Search History** | `Ctrl+R` |
| **Stop Process** | `Ctrl+C` |

## Chrome DevTools

| Action | macOS | Windows/Linux |
|---|---|---|
| **Open DevTools** | `Cmd+Opt+I` | `Ctrl+Shift+I` |
| **Toggle Device Mode** | `Cmd+Shift+M` | `Ctrl+Shift+M` |
| **Search File** | `Cmd+P` | `Ctrl+P` |
