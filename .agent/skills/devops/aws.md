# AWS Skills

## Core Services
- **EC2**: Virtual servers.
- **S3**: Object storage (images, logs, static sites).
- **RDS**: Managed relational databases (Postgres, MySQL).
- **Lambda**: Serverless functions.
- **API Gateway**: Expose Lambda functions as HTTP APIs.
- **DynamoDB**: NoSQL database.

## IaC (Infrastructure as Code)
- Use **Terraform** or **CDK** (Cloud Development Kit).
- Avoid manual console clicks for production resources.

## Security
- **IAM**: Principle of Least Privilege.
- **Security Groups**: White-list IPs/Ports.
- **VPC**: Isolate resources in private subnets.
