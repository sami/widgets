# Android Kotlin Skills

## Modern UI (Jetpack Compose)
- **Declarative**: Use Compose for UI.
- **State Hoisting**: Move state up to make components reusable.

## Architecture
- **MVVM**: Model-View-ViewModel.
- **Clean Architecture**: Domain, Data, Presentation layers.

## Kotlin Features
- **Coroutines**: Use for async work (`suspend` functions).
- **Flow**: Cold streams for reactive programming.
- **Data Classes**: Use for holding data.

## Jetpack Libraries
- **Room**: Local database.
- **Hilt/Dagger**: Dependency injection.
- **Navigation**: Typed navigation.
- **ViewModel**: Lifecycle-aware UI data holder.
