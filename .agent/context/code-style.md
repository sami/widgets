# Code Style Guide

## General
- Indentation: 2 spaces (JS/TS/HTML), 4 spaces (Python).
- Max Line Length: 80-100 characters.
- File Names: `kebab-case` (folders/files) or `PascalCase` (components).

## JavaScript/TypeScript
- Use `const` over `let`. Avoid `var`.
- Use arrow functions.
- semicolons: true.

## Python
- Follow PEP 8.
- Use `snake_case` for functions/variables.
- Use `PascalCase` for classes.
- Type hints are mandatory.
